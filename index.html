A few projects I keep coming back to when people ask what I actually build.

WinePair

A production pairing site used by restaurants and diners. It matches dishes to wines in real time based on each restaurant’s inventory. I worked on the full-stack rebuild with a FastAPI backend, Azure SQL, and a React + Tailwind frontend, then tightened up the pairing pipeline so results felt instant instead of “go make tea and come back.”

Honeycomb

A disaster messaging network that keeps going even when nodes drop or connections get weird. It is built on a Chord-inspired ring, and we stress tested it on a 70-node EC2 setup to see how fast it recovered and how messages moved under failures.
